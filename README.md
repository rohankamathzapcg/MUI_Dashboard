# React Material-UI Application

This project is a responsive and visually appealing UI created using React and Material-UI (MUI). The application includes a `Create Post Modal`, `Sidebar`, `Right Bar`, and a `Feed` section. It also features a **Light and Dark mode toggle** to improve usability.

## Features

- **Create Post Modal**: A modal dialog to add new posts.
- **Sidebar Navigation**: A collapsible sidebar for easy navigation.
- **Right Bar**: Additional sections for user details or trending topics.
- **Feed Section**: Displays a list of posts.
- **Light and Dark Mode**: A toggle to switch between light and dark themes.

## Built With

- **React**: A JavaScript library for building user interfaces.
- **Material-UI (MUI)**: A popular React UI framework for creating responsive and accessible designs.

## Components Used

1. **Box**: For layout and structure.
2. **Stack**: For vertical and horizontal stacking of elements.
3. **AppBar** and **Toolbar**: For the header/navigation bar.
4. **Tooltip**: For providing additional information on hover.
5. **Button**: For various interactive actions.
6. **Modal**: For the "Create Post" functionality.

## Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/react-mui-ui.git
```
2. Navigate to the project directory:
```bash
cd react-mui-ui
```
3. Install dependencies:
```bash
npm install
```
4. Start the development server
```bash
npm start
```
